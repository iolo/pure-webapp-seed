pure-webapp-seed
================

My own starting point for writing pure-webapp(runs on browser/front-end/client-side).

Please, raise issues to to keep up-to-date.

Ingredients
-----------

* angular 1.0.5
* bootstrap 2.3.1
* google-code-prettify 4-Mar-2013
* html5shiv 3.6.2
* jquery 1.9.1 with plugins
